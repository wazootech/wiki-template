# Hello world

Welcome to your new semantic wiki!

This is a basic "Hello World" document to get you started. The `llm-wiki` package (`wiki` command) enables you to manage documents, validate them with SHACL, query them with SPARQL, and publish them as a static site.

To explore further:
- Modify this file to add content.
- Create new `.md` files in this directory.
- Add SHACL shapes to define constraints for your document types.

Enjoy building your semantic knowledge base!
